# Market Tavern — SEO Visual Demo

A visual demonstration showing Market Tavern (Stockton, CA) how their dishes would appear in Google search results with a modern, indexable website vs their current PDF menu.

## Purpose

Used during cold call pitch to show Chef Nicholas DiArenzo / Phillip Wong how their restaurant is invisible to Google right now, and how a modern website would change that.

## Features

- Simulated Google search results (before/after toggle)
- 6 real search queries their customers are making
- Visual comparison: PDF menu vs modern indexable site
- Shows Market Tavern ranking #1 after new site launch
